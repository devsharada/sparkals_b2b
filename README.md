# sparkals_b2b

💎 A sleek and responsive gemstone showcase website crafted with Angular and Tailwind CSS. Designed with a modern UI approach, this project highlights elegant product presentation, smooth user experience, and scalable frontend architecture.

🚀 Features
    ✨ Fully responsive design using Tailwind CSS
    💎 Beautiful gemstone product listings
    🔍 Search and filter functionality
    🛒 Shopping cart UI (optional if implemented)
    ⚡ Fast and optimized Angular architecture
    🎨 Clean and modern UI/UX

🛠️ Tech Stack
    Angular – Frontend framework
    Tailwind CSS – Utility-first styling
    TypeScript – Strongly typed JavaScript
    HTML5 & CSS3
    
📌 Purpose
This project was built to practice frontend development skills, focusing on component-based architecture, responsive design, and modern UI styling.
